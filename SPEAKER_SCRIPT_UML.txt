SLIDES 8-14: DESIGN DECISIONS & TRADE-OFFS
===============================================================================

DESIGN DECISION #1: Municipality Lookup
"Option A: Exact string match (fast but fragile)
Option B: Geocoding + fuzzy matching ✓ (robust, handles typos, TTL cache)"

DESIGN DECISION #2: PV Computation
"Option A: Call Google API for all 50k buildings = €1,500 + 8 hours ❌
Option B: Pre-filter to 50, call Google only for winners = €1.50 + 2-3 min ✓"

DESIGN DECISION #3: Where AI Fits
"✓ AI: Address fuzzy matching (NLP)
✓ AI: Writing narratives (LLM)
❌ AI: Financial calculations (LLMs hallucinate numbers, risk is too high)
Solution: AI writes text only, all numbers from deterministic code"

DESIGN DECISION #4: CRM Join
"❌ Option A: Exact address match (brittle)
✓ Option B: Fuzzy NLP + coordinate proximity (robust)"

V1 SHIP STRATEGY (2 Weeks):
"CUT: Full pre-computation (do on-demand instead)
CUT: Bi-directional CRM sync (read-only for now)
CUT: AI agent loops (single LLM call, simple)
KEEP: Core insight (pre-filter → selective API calls)

If this saves sales engineers 3 hours per municipality, we have product-market fit."

===============================================================================
SLIDE 15: SECURITY & RISK
===============================================================================
"Production safeguards:
- OAuth2 authentication (HubSpot login)
- TLS 1.3 encryption (data in transit)
- AWS KMS encryption (data at rest)
- Rate limiting (1000 req/min per user)
- Circuit breaker (graceful degradation if APIs fail)

AI risk mitigation:
- Pydantic schemas validate every output
- No AI-generated numbers (only text)

Compliance:
- GDPR: 24-month data retention
- Audit logs: Every API call recorded
- SOC2 ready: CloudWatch + Datadog monitoring"

===============================================================================
SLIDE 16: SUMMARY - THE CORE INSIGHT
===============================================================================
"You can't compute 50,000 buildings in 2 minutes.

But you CAN:
1. Pre-filter to 5,000 via static geometry (fast)
2. Rank by roof size (highest ROI first)
3. Call expensive API only for top 50
4. Join with CRM data via fuzzy matching
5. Generate narrative with LLM (text only)

Result: 2-3 minutes, €1.50 cost, sales-ready briefing

Every design decision traces back to this insight."

KEY PRESENTATION TIPS
===============================================================================
- Speak naturally, don't read the script
